# ferust (btw rusta yazdım)
minimal hızlı ve ekstra kütüphane kullanmadan yazdığım cihazın bilgisini getiren araç

## Yükelemek için: 
- `cargo build --release && cd target/release && chmod +x ferust && sudo mv ferust /usr/local/bin`
